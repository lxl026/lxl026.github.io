《C++ Primer 第三、四章》读书笔记

  • 命名空间

    • 使用作用域符 std::cout
    • using 声明 using std::cout
    • using 指示 using namespacce std

  • 头文件不要包含using声明

  • 直接初始化：不使用等号初始化一个对象
    拷贝初始化：使用等号初始化一个对象；将一个对象作为实参传递给非引用类型的形参；从一个返回类型为非引用类型的函数返回一个对象；用花括号初始化一个数据或struct

  • 理解复杂数组声明
    有括号：从内到外
    无括号：从右到左

  • 运算符重载可重载运算符对象和返回值类型，但是不可以改变运算对象的个数、运算符的优先级以及结合律

  • 左值和右值
    当一个对象被用作右值时，用的是对象的值（内容）
    当一个对象被用作左值时，用的是对象的身份（在内存中的位置）

  • 求值顺序是不确定的，所以如果改变了某个运算对象的值，在表达式的其他地方就不要再使用这个运算对象

  • 尽量不要用递增递减运算符的后置版本

  • sizeof运算符并不实际计算其运算对象的值，比如：
    sizeof(++i)并不会让i递增，也可以通过某个类型的对象的空指针计算该类型的大小

  • 类型转换

    • 隐式转换
      整型提升
      数组转指针
      在if语句中，指针转bool

    • 显式转换

      • 命名的强制类型转换：cast-name<type>(expression)
        static_cast：任何具有明确意义的转换（不包括底层const）
        dynamic_cast：支持运行时类型识别
        const_cast：去掉const
        reinterpret_cast：低层次上的重新解释
      • 旧式的强制类型转换
        type(expr)
        (type)expr
        如果const或static转换合法就用，不合法就执行与reinterpret_cast类似的功能
